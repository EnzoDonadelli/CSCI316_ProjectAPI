# Photo Client

This is a minimal Vite + React + TypeScript scaffold that connects to the VisaoAPI backend.

Quick start

1. Open a terminal in this folder (Client/photo-client)
2. Install dependencies: npm install
3. Start dev server: npm run dev

By default the client will call the API base URL from the environment variable `VITE_API_URL`. If not provided it falls back to `http://localhost:5000`.

Example (PowerShell):

    cd "Client/photo-client"; npm install; npm run dev
